# Project Overview

The **AI Workplace Productivity Assistant** is a frontend-only web application designed to improve workplace productivity using Artificial Intelligence. It provides users with AI-powered tools to summarise meeting notes, organise tasks, and interact with an AI workplace assistant through an intuitive dashboard.

The application features a clean, modern SaaS-inspired user interface that is fully responsive across desktop, tablet, and mobile devices. Since it is designed as a lightweight productivity tool, it does not require user registration, login, or a database. All user interactions occur within the current browser session without storing any personal data.

# Setup Instructions

## Prerequisites

Ensure you have the following installed:

- A modern web browser (Google Chrome, Microsoft Edge, Firefox, or Safari)
- Git (optional, for cloning the repository)

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-workplace-productivity-assistant.git
```

### 2. Navigate to the project folder

```bash
cd ai-workplace-productivity-assistant
```

### 3. Open the application

Open the project in your preferred code editor or simply launch the `index.html` file in your web browser.

Alternatively, if using Visual Studio Code, install the **Live Server** extension and run the project locally.

## Usage

1. Open the application.
2. Select a feature from the sidebar.
3. Enter your input (meeting notes, tasks, or chatbot prompt).
4. Generate AI-powered content.
5. Edit the generated output as needed.
6. Copy or use the content for your workplace tasks.

> **Note:** This application is frontend-only. It does not require user authentication, does not connect to a backend server, and does not store user data.
